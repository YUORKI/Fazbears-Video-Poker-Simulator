# FNAF BALATRO (placeholder)

**THIS MOD IS CURRENTLY IN DEVELOPMENT**

A Balatro mod that aims to add FNAF themed content to balatro

Currently, this mod adds:
- 24 Jokers
- 2 Tarot Cards
- 3 Enhanced Cards
- 2 Seals
- 1 consumable types:
- - 3 Item cards : used for utility during rounds

## Installation

This mod requires both [Lovely](https://github.com/ethangreen-dev/lovely-injector) and [Steammodded](https://github.com/Steamopollys/Steamodded) **(Version 1.0.0-beta-0614a or later)**.

Afterwards, simply drag the files from the latest release and put it in your Mods folder (`%APPDATA%/Balatro/Mods`). 

***IMPORTANT NOTE:*** **THIS MOD IS CURRENTLY IN DEVELOPMENT meaning it has bugs and content that is being worked on**
